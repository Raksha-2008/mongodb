# Assignment API (Express + MongoDB)

## Features
- Create Assignment (POST)
- Get All Assignments (GET)
- Update Assignment (PUT)
- Delete Assignment (DELETE)

## Technologies Used
- Node.js
- Express.js
- MongoDB

## API Endpoints

POST /assignments  
GET /assignments  
PUT /assignments/:id  
DELETE /assignments/:id
